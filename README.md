## 需求分析
> 定位与目标：利用c++程序语言设计一款游戏，要求游戏操作简单，游玩愉快有趣。  
> 平台：funcode  
> 操作系统： win7x86位操作系统  
> 面向群体：普通大众  
> 性能需求：在设想的标准系统下的相应时间不应有超过0.2s的延迟
